Configuración de Firebase
⚙️ Paso a paso para activar la base de datos:

1. Ve a console.firebase.google.com → Crea un proyecto gratis
2. En el proyecto → Firestore Database → "Crear base de datos" → Modo producción
3. Ve a Configuración del proyecto → "Tus apps" → Agrega app web → Copia las credenciales
4. Abre el archivo prueba_candidato.html con un editor de texto
5. Busca la sección 🔧 CONFIGURACIÓN FIREBASE y reemplaza los valores
6. Haz lo mismo en este archivo panel_admin.html
7. Sube ambos archivos a Firebase Hosting (gratis) o cualquier hosting

¿Sin hosting propio? Puedes usar Netlify.com o Vercel.com — ambos gratuitos, arrastra y suelta los archivos.
